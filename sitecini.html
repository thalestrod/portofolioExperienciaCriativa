Site Cinema

Para realizar este trabalho, utilizamos a plataforma Visual Studios. Ultilizamos as linguagens HTML, CSS e Java Script.
O objetivo do trabalho era ultilizar as linguagens acima para implementar videos, imagens, texto e cadastro. Optamos por um
site de cinema, onde a pessoa tem acesso aos cartazes, trailers e informações sobre o filme.
